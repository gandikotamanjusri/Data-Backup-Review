# Data-Backup-Review

# Data Backup System (Java Console App)

A menu-driven Java application to create, store, restore, compress, and manage backup data using:

Binary files (.dat)

Compressed files (.dat.gz)

CSV files (.csv)

Version management

Cleanup and statistics

# Project Structure

DataBackupSystem/
│
├── BackupData.java
├── BackupManager.java
├── FileUtils.java
├── CompressionUtils.java
├── CSVHandler.java
├── VersionManager.java
├── backups/              (auto-created)
└── README.md
